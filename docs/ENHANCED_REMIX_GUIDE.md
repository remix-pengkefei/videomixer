# 超强混剪指南

## 更新时间: 2025-02-04 21:10

---

## 核心原则

### 过审要点
1. **大量视觉改动** - 让平台算法认为是新内容
2. **贴纸数量充足** - 至少12-20个
3. **遮罩面积足够大** - 顶部200px+，底部220px+
4. **多层渐变过渡** - 避免生硬边界
5. **色块装饰丰富** - 左右两侧 + 中间区域
6. **闪烁粒子** - 50个左右增加动态感

## 三种视频风格

**适用**: 手写文字、打字效果、文案类视频

**配色方案**:
- 遮罩: `#1a1a2e` (深蓝黑)

- 蕾丝花边
- 古董边框和装饰品

## 批量处理示例

```python
import os
from pathlib import Path

# 处理整个文件夹
input_dir = "/path/to/videos"
output_dir = "/path/to/output"

for video in Path(input_dir).glob("*.mp4"):
    output = Path(output_dir) / f"{video.stem}_mixed.mp4"
    os.system(f'python -m src.enhanced_handwriting "{video}" "{output}"')
```

## 注意事项

1. **贴纸要用19000素材库** - `assets/stickers/`里的动物贴纸太简单
2. **遮罩要够大** - 小于200px容易被识别为原视频
3. **渐变很重要** - 多层渐变让效果更自然
4. **粒子要够多** - 50个左右才有足够的视觉变化
5. **处理时间** - 1分钟视频约需30秒处理
