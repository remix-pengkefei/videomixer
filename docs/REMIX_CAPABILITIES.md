# VideoMixer 混剪能力清单

> 基于 ProcessedVideos 文件夹中人工制作的优秀混剪视频分析总结
> 分析日期: 2025-02-03

---

## 一、视频类型分类

### 1. 数字人视频 (Digital Human)
- AI生成的说话人物视频
- 特点：口型同步、表情自然、背景可替换

### 2. 手写文字视频 (Handwriting Animation)
- 模拟手写/打字的文字动画
- 特点：逐字显示、毛笔/钢笔效果

### 3. 音乐播放器视频 (Music Player Style)
- 模拟音乐APP界面风格
- 特点：专辑封面、播放控制、进度条

### 4. 游戏录屏混剪 (Gaming Remix)
- 游戏画面+文字解说混剪
- 特点：多画面拼接、弹幕风格

### 5. 情感类视频 (Emotional Content)
- 配合抒情音乐的图文视频
- 特点：精美背景、歌词同步

## 四、视频类型与策略对应

### 数字人视频策略
```
推荐元素:
- 字幕: S1/S2 (底部横排+描边)
- 贴纸: D1/D2 (角标+节日主题)
- 特效: P1 (闪光粒子)
- 背景: 保持原背景或B4替换

覆盖率: 30-50%
装饰风格: 简洁为主
```

### 手写文字视频策略
```
推荐元素:
- 标题: T2/T3 (3D立体+书名号)
- 贴纸: D4/D5 (可爱+图标组)
- 背景: 纸张/牛皮纸纹理
- 特效: P2 (飘落粒子)

覆盖率: 60-80%
装饰风格: 丰富热闹
```

### 音乐播放器视频策略
```
推荐元素:
- UI: U1/U3 (播放器+进度条)
- 背景: B1/B2 (动态+水波纹)
- 贴纸: D2/D3 (节日+3D立体)
- 标题: T1/T5 (金色+手写)

覆盖率: 70-90%
装饰风格: 精美华丽
```

### 游戏录屏视频策略
```
推荐元素:
- 布局: L2 (上下分割)
- 贴纸: D2/D4 (节日+可爱)
- 字幕: S8 (多行大字)
- UI: U4 (REC标识)

覆盖率: 50-70%
装饰风格: 活泼动感
```

### 情感类视频策略
```
推荐元素:
- 背景: B2/B6 (水波+墨迹)
- UI: U1 (音乐播放器)
- 贴纸: D2/D3 (节日+3D)
- 字幕: S7 (歌词同步)

覆盖率: 60-80%
装饰风格: 精美抒情
```

## 六、技术实现建议

### FFmpeg滤镜对应

```python
# 水波纹效果
"geq=lum='lum(X,Y)':cb='cb(X,Y+sin(T*2)*5)':cr='cr(X,Y+sin(T*2)*5)'"

# 粒子叠加
"movie=particles.mov[p];[v][p]overlay=shortest=1"

# 画中画
"[0]scale=iw/2:-1[pip];[1][pip]overlay=W-w-10:H-h-10"

# 模糊边框
"split[a][b];[a]crop=iw*0.8:ih:(iw-iw*0.8)/2:0[center];[b]boxblur=20[blur];[blur][center]overlay=(W-w)/2:0"

# 文字逐字显示
"drawtext=text='文字':fontsize=60:x=(w-tw)/2:y=h-100:enable='gte(t,0)*lt(t,2)'"
```
